# v0.2 — Generative b-roll (clip tier) — first increment

**Status:** plan
**Source:** ROADMAP 0.2 + ClipApiResearch brief (Kling new single-key API; Veo needs a Gemini API key)
**Goal:** get a real generated clip into a rendered scene. Kling first (we have a direct key);
Veo deferred until a Gemini API key exists.

## WI-6 — Kling clip adapter
`providers/clip/kling.mjs` implementing the `clip` contract via Kling's official API.
- **AC6.1** stdin `{brief, aspect, duration_s, ref_frame?, out_path}` → stdout `{clip_path, duration_ms}`.
- **AC6.2** Auth: `Authorization: Bearer <key>` from `.video-factory/config.json` (`clip.kling_api_key`) / env; never args.
- **AC6.3** Submit `POST /v1/videos/text2video` (`model_name: kling-v2-6`, prompt=brief, duration "5"/"10" from duration_s, aspect_ratio, mode "std"); if `ref_frame` given, use `/v1/videos/image2video` with the frame as `image`.
- **AC6.4** Poll `GET …/{task_id}` every ~10s until `succeed`/`failed` (timeout ~10min); on succeed, **download `task_result.videos[0].url` immediately** (URL expires) to `out_path`.
- **AC6.5** Errors clean to stderr + non-zero exit: insufficient credits (surface the `required` vs `available` if present), failed task (`task_status_msg`), timeout. JSON-only stdout.
- **AC6.6** `duration_ms` measured via ffprobe on the downloaded file.

## WI-7 — B-roll scene end-to-end
Prove a `visual.source:"clip"` scene renders real footage.
- **AC7.1** A test manifest with one clip scene (semantic `visual_brief`) → producer calls the clip adapter → writes `visual.asset` + measured `duration_ms`.
- **AC7.2** Compositor stages the clip into `public/` and renders it (`OffthreadVideo`); a frame extract shows the generated footage (not a code card).
- **AC7.3** Mixed manifest (code + clip scenes) renders coherently in one MP4.

## Test strategy

success). WI-7 renders a real clip scene and verifies a frame is footage. Costs: one Kling

## Risks
- **Kling credits unknown** → first call may return insufficient-credits; that's a finding, not a failure.
- **Key may be a reseller key** → if `api.klingai.com` 401s, try `api-singapore.klingai.com` / note reseller base URL.
- **Clip URL expiry** → download immediately on poll success.
